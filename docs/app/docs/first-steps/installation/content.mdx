# Installation

Getting started with Rad UI is simple and quick! Whether you're integrating it into an existing project or starting fresh, you'll be up and running in no time. Follow the instructions below to install Rad UI using your preferred package manager.

## Install via npm


If you use npm, run the following command to install Rad UI:

<MultipleTabs items={[
  { manager: 'pnpm', command: 'pnpm install @radui/ui' },
  { manager: 'npm', command: 'npm install @radui/ui --save' },
  { manager: 'yarn', command: 'yarn add @radui/ui' },
  { manager: 'bun', command: 'bun add @radui/ui' },
]} />
